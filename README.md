# MiPlan

一款面向个人的 **日历 + 待办** 桌面应用：在月视图上管理每日事项，数据默认保存在本机（Windows 可配合 SQLite），无需登录即可使用。当前版本聚焦 **本地开发与小范围分发**；云端账号与同步留待后续迭代。

> 本仓库可为 **私有**，不表示代码对外开源；README 用于说明产品与安装方式。

---

## 功能概览

- **月历视图**：按年月浏览，支持滚轮快速切换月份；格子展示各标签下未完成数量。
- **单日待办**：点击日期打开侧栏，添加 / 完成 / 删除事项，支持 **标签** 与 **单日去重**（相同文案与标签的未完成项不重复添加）。
- **批量添加**：在「批量添加」中填写事项与标签，在迷你日历上点选或拖选多日，一次性写入；已选格可再点取消；批量写入规则与单日一致。
- **标签**：内置多类标签，支持自定义标签与颜色排序（网页预览下存于浏览器存储，桌面版存于 SQLite）。
- **清空本月 / 清空本日**：按月或按日快速清空待办（本月清空带确认，本日无二次确认）。
- **个性与模式**：昵称、自定义头像（本地图片）；设置中可查看「未登录 / 已登录」占位状态，以及 **账号管理** 下的「清空本地数据」（重置本机库与浏览器相关存储）。
- **桌面端**：Tauri 封装，待办与标签使用本地 **SQLite**；可构建 **Windows 安装包**（如 NSIS `.exe`）。

---

## 技术栈（简要）

- 前端：React、TypeScript、Vite、Tailwind CSS  
- 桌面：Tauri 2、SQLite（`tauri-plugin-sql`）  
- 默认 **无需服务器** 即可使用本地功能

## 安装方式

### 推荐：从 GitHub Releases 下载

1. 打开本仓库的 **Releases** 页面。  
2. 在最新版本中找到 **Windows 安装程序**（例如 `MiPlan_x.x.x_x64-setup.exe`）。  
3. 下载后运行安装包，按向导完成安装。

## 数据与隐私

- 待办、标签、个性设置等默认仅保存在 **本机**（桌面版为 SQLite + 少量本地配置；网页预览为浏览器 `localStorage`）。  
- **当前版本不包含** 账号注册、云端同步与自动上传；后续若接入，会在 README 中单独说明。
